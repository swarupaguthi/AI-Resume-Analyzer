# AI-Resume-Analyzer
AI-powered resume analyzer for tech internships. Instant ATS scoring, pie chart breakdowns (Skills/Exp/Edu), detects 16+ languages (Python/React/SQL++). Top 5 priority skills w/ YouTube links, 30-day internship roadmaps, pro ATS tips.
Smart Resume AI Pro works in 4 simple steps: First, you upload any PDF resume and PyPDF2 instantly extracts all text. Second, it scans 60+ keywords across 16+ languages (Python, React, SQL, AWS, Docker) to calculate your ATS score using Skills(30%) + Experience(35%) + Education(25%). Third, Matplotlib generates a compact pie chart showing the exact breakdown. Finally, based on your score, it prescribes top priority skills ("Python NOW" if <70%, "LeetCode 300" if 85%+) plus a personalized 30-day internship roadmap.
